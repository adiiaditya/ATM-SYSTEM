# ATM-SYSTEM
This project was created as a part of coursework at Illinois Institute of Technology at Chicago.


TO RUN IN COMMAND PROMPT/ TERMINAL ON MAC
=============================================
java -jar ATM.jar


TO COMPILE & RUN
================

1. go to “ATM” folder
2. go to “src” folder
3. compile all .java files with
	javac *.java
4. run with
	java Driver
